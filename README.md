# Rock Paper Scissors 🎮

A console-based Rock Paper Scissors game built with Python.

## Description

Interactive command-line game where the user plays against the computer. 
The computer's choice is randomly generated, and the winner is determined 
by classic Rock Paper Scissors rules.

## Features

- User input validation
- Random computer move generation
- Win/loss/tie detection
- Clean console output

## Technologies

- Python 3
- `random` module
- Conditional logic & control flow

## How to Run
```bash
python rock_paper_scissors.py
```

- Implementing game logic with conditional statements
- Handling user input and edge cases
- Using Python's `random` module for computer behavior
